Karamoja Crop Production Data Analysis
Project Overview

This project analyzes maize and sorghum production in the Karamoja region of Uganda at both district and subcounty levels. The analysis explores production patterns, population influence, land use efficiency, and yield distribution using Python data science tools.

The workflow includes:

Extracting datasets

Cleaning and transforming data

Merging datasets

Feature engineering

Data visualization

Exporting processed data

Data Integration

The district and subcounty datasets are merged using:

District

karamoja_df

Feature Engineering

New analytical features are created:

1. Productivity Per Capita
Total Production ÷ Population

2. Population Density
Area ÷ Population

3. Land Use Efficiency
(Crop Area ÷ Total Area) × 100


These features help evaluate agricultural performance and land utilization.

📈 Key Findings
Production

Maize production is higher than sorghum across most districts.

Kotido shows relatively high sorghum production.

Moroto has low production for both crops.

Yield Distribution

Maize yields show slight negative skew.

Sorghum yields show slight positive skew.

Correlation

Maize and sorghum yields are positively correlated.

Crop area has weak correlation with yield.

Productivity depends more on environmental and management factors.
